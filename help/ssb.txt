Overview of Secure-Scuttlebutt (SSB) Concepts 

See the protocol guide for more details 
(https://ssbc.github.io/scuttlebutt-protocol-guide).

SSB_ID: Log/Feed/Author identifier (SSB ID), which represent the public 
key of a public-private key pair used to authenticate messages.
    (ex: "@IgYpd+tCtXnlE2tYX/8rR2AGt+P8svC98WH3MdYAa8Y=.ed25519")

SSB_MSG_ID: Reference to a message (Message ID)
    (ex: "%b/UUNV5VgLLxyt+yDMKomLqYIdSF4rMJBVRXO3GPFjQ=.sha256").
