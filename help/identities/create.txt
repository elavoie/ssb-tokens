usage: ssb-tokens identities create [<options>]

Create a new ssb identity (SSB_ID).

<options> (default?, default value, type):

    --alias=NAME (false, null, String)
        Associates NAME to the newly created identity.
        (See 'ssb-tokens identities alias')

See 'ssb-tokens help ssb' for an explanation of the Secure-Scuttlebutt 
concepts (ex: SSB_ID) used.
